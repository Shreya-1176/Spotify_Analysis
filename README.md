# 🎵 My Spotify Listening Analysis

A personal data analysis project using my own Spotify streaming history data.

## About
Analyzed 29,535 songs (23,664 after cleaning) from my personal Spotify account to uncover listening patterns and habits.

## Key Findings
- 🎧 Total listening time: **1,210 hours**
- 👑 Top artist: **BTS (89 hours)**
- 🎵 Most played song: **"I Like Me Better" by LAUV (89 plays)**
- 🦉 Peak listening time: **3am - 5am**
- 📅 Highest listening month: **August**

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab
